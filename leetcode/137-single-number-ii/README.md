给定一个**非空**整数数组，除了某个元素只出现一次以外，其余每个元素均出现了三次。找出那个只出现了一次的元素。

**说明：**

你的算法应该具有线性时间复杂度。 你可以不使用额外空间来实现吗？

**示例 1:**

```
输入: [2,2,3,2]
输出: 3
```

**示例 2:**

```
输入: [0,1,0,1,0,1,99]
输出: 99
```

#### 思路
1. 计算每bit位上1的数量和，对其%3，结果为0或1（不会为2），拼接起来即可得到最终结果。

2. 使用两个值once,twice表示状态
once=0,twice=0 初始状态
once=1,twice=0 出现1次
once=0,twice=1 出现2次
once=0,twice=0 出现3次
